Workflow Git Demo

Version 0.16.2

À propos

Ce projet démontre l'implémentation d'un workflow Git moderne avec GitFlow, Release Please, et validation automatique des commits conventionnels.

Fonctionnalités

  • GitFlow complet avec branches main et develop
  • Release Please pour gestion automatique des versions
  • Validation des Conventional Commits via Husky
  • PR Guard pour protection des branches
  • Changelog automatique avec emojis
  • Support des hotfix et release branches

Types de Commits

feat: Nouvelle fonctionnalité fix: Correction de bug hotfix: Correction urgente chore: Maintenance docs: Documentation style: Formatage refactor: Refactorisation test: Tests

Format de Commit

type(scope): description

Exemple: feat(auth): ajouter authentification OAuth

Exemple: feat(user): on ajoute une authentification OAuth à l'user
